# Contributing to Defendly

Defendly is a fork of OWASP ZAP, customized for enterprise use. While this is a private fork, contributions are managed internally.

## Building Defendly

To build Defendly from source, refer to [BUILDING.md](BUILDING.md).

## Reporting Issues

For issues specific to Defendly, please contact your internal development team.
